About DSU Examination System

A professional database-driven online examination portal for DHA Suffa University.

Project Overview

This system helps students register, login, view available exams, attempt MCQ-based tests, submit answers, and view results with grades and performance analytics. Faculty/admin can add exams, add questions, delete exams, and monitor student results using MySQL database integration.
