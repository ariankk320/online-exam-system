# Online Examination System

This project is developed as a **4th semester Database Lab project** for DHA Suffa University.  
It is a full-stack Online Examination System using **Node.js, Express, MySQL, HTML, CSS, and JavaScript**.

---

## Features

- **Student Registration & Login**
- **Admin Login**
- **Exam Creation**
- **MCQ Management**
- **Online Exam Attempt**
- **Result Management**
- **Excel/CSV Result Export**
- **Dashboard for Admin and Students**
- **Responsive Frontend**

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Other Tools:** Git, Railway (for deployment)
